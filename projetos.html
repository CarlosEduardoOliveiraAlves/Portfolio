Calculadora Simples

Realizar operações matemáticas básicas (adição, subtração, multiplicação e divisão).

Este projeto é uma calculadora web que permite aos usuários realizar cálculos simples. A interface é intuitiva, com botões para números e operações, além de um campo de exibição que mostra os resultados. A calculadora é desenvolvida utilizando HTML, CSS e JavaScript.


To-Do List

Permitir aos usuários adicionar, marcar como concluídas e remover tarefas.

A To-Do List é uma aplicação web que ajuda os usuários a organizarem suas tarefas diárias. Os usuários podem adicionar novas tarefas, marcar as concluídas e removê-las da lista. O projeto é uma ótima oportunidade para praticar manipulação do DOM com JavaScript e estilização com CSS.


Galeria de Imagens

Exibir uma coleção de imagens em um layout responsivo.

Este projeto consiste em uma galeria de imagens que se adapta a diferentes tamanhos de tela. Os usuários podem visualizar miniaturas das imagens e clicar nelas para exibir uma versão maior. A galeria é construída usando HTML, CSS e JavaScript para a funcionalidade de ampliação.
